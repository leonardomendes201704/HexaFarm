# EPICO-016 - Ambientacao Visual da Tela Inicial

## Tipo

Epico

## Objetivo

Melhorar a imersao da home aplicando uma arte de fundo que preencha toda a tela inicial.

## Problema que Este Epico Resolve

A tela inicial ainda dependia de fundos abstratos e paines, sem usar a arte principal do projeto como camada de ambientacao total.

## Valor de Negocio

- reforca identidade visual logo na entrada
- aumenta a sensacao de jogo antes mesmo da run comecar
- melhora a apresentacao da home com uma arte hero real

## Resultado Esperado

A tela inicial passa a usar uma imagem de fundo cobrindo 100% da viewport, com os elementos do menu por cima.
