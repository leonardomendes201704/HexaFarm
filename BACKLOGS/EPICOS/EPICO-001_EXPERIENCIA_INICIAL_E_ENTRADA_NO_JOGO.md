# EPICO-001 - Experiencia Inicial e Entrada no Jogo

## Tipo

Epico

## Objetivo

Estabelecer a experiencia inicial de entrada do jogador em `HexaFarm`, garantindo uma primeira impressao consistente com a identidade cozy/cute do projeto e uma navegacao inicial clara no browser.

## Problema que Este Epico Resolve

Sem uma tela de entrada clara e bem estruturada, o jogo perde identidade logo no primeiro contato e fica sem um ponto de acesso consistente para novas runs, continuidade de progresso e configuracoes basicas.

## Valor de Negocio

- cria a primeira impressao do produto
- estabelece identidade visual inicial
- define o ponto de entrada padrao da aplicacao
- prepara a base para futuras telas e navegacao

## Resultado Esperado

Ao abrir o jogo, o usuario deve visualizar imediatamente uma tela inicial clara, estilizada e navegavel, com acoes para iniciar uma nova partida, continuar uma sessao salva e acessar opcoes basicas.

## PBI Vinculado

- `PBI-001 - Tela Inicial do Jogo HexaFarm`

## PBIs Futuros Naturais

- `PBI-002 - Estrutura base de navegacao entre telas`
- `PBI-003 - Sistema minimo de save local`

## Criterios de Sucesso do Epico

1. O jogo possui uma tela inicial funcional como entrada padrao.
2. A identidade de `HexaFarm` fica clara nos primeiros segundos de uso.
3. O fluxo de novo jogo e continuidade de save fica preparado para evolucao.
4. A base tecnica permite acoplar novas telas sem retrabalho estrutural relevante.

## Dependencias

- estrutura base do app React
- tela raiz do aplicativo
- mecanismo simples de leitura de save local ou stub equivalente

## Definicao de Pronto

Este epico estara concluido quando o `PBI-001` estiver entregue e validado conforme seus criterios de aceitacao.
