# EPICO-007 - Deckbuilding Persistente e Economia da Run

## Tipo

Epico

## Objetivo

Transformar o prototipo em um loop de run mais proximo de um deckbuilder incremental, com deck configuravel, pilhas reais de compra e descarte, e uma economia de aluguel/loja entre runs.

## Problema que Este Epico Resolve

O prototipo atual ja tem cartas, energia e fim de dia, mas ainda nao possui um deck real montado pela jogadora nem uma economia de progressao entre runs.

## Valor de Negocio

- aumenta o controle estrategico da jogadora sobre o baralho
- cria metaprogressao por colecao de cartas
- adiciona objetivo economico claro para cada run

## Resultado Esperado

Cada run comeca com a montagem de 24 cartas da colecao possuida, dura 7 dias, termina com validacao de aluguel e abre uma loja para expandir a colecao antes da proxima run.

## Criterios de Sucesso do Epico

1. A run passa a usar um deck configurado pela jogadora.
2. As pilhas de compra e descarte ficam legiveis na interface.
3. A run possui duracao fechada de 7 dias.
4. Existe aluguel progressivo e loja ao fim da run.
