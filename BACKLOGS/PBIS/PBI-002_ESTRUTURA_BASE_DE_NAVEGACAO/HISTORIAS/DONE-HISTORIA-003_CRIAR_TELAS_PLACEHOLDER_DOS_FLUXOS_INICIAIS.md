# HISTORIA-003 - Criar Telas Placeholder dos Fluxos Iniciais

## Tipo

Historia

## Vinculo

- Epico: `EPICO-001`
- PBI: `PBI-002`

## Declaracao

Como jogador, quero visualizar telas dedicadas para os fluxos iniciais, para perceber que o jogo ja possui uma estrutura de navegacao pronta para evoluir.

## Criterios de Aceitacao

1. Existe uma tela placeholder para `Novo Jogo`.
2. Existe uma tela placeholder para `Continuar`.
3. Existe uma tela placeholder para `Opcoes`.
4. Todas permitem retorno para a home.
