# TASK-008 - Representar Cultivos com Props 3D ou Billboards

## Status

Concluido

## Objetivo

Fazer `Milho`, `Abobora` e `Tomate` aparecerem visualmente sobre o tile plantado.

## Implementacao

- adicionados props 3D simples para cultivos no stage
- `Milho`, `Abobora` e `Tomate` agora possuem marcadores 3D proprios
- o stage 3D passa a refletir visualmente quando um tile tem cultura plantada

## Resultado Esperado

Tiles com cultivo passam a mostrar uma leitura visual clara da cultura plantada diretamente no mapa 3D.
