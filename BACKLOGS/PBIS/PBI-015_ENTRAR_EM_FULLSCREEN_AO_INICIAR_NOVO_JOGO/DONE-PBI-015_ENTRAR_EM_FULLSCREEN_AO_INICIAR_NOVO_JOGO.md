# PBI-015 - Entrar em Fullscreen ao Iniciar Novo Jogo

## Tipo

PBI (Product Backlog Item)

## Vinculo

- Epico pai: `EPICO-014 - Imersao de Entrada da Run`

## Titulo

Colocar o jogo em tela cheia ao iniciar uma nova run

## Objetivo

Usar a API de fullscreen do navegador no clique de `Novo Jogo`, antes de navegar para a tela principal da run.

## Criterios de Aceitacao

1. O clique em `Novo Jogo` tenta entrar em tela cheia.
2. A navegacao para a run continua funcionando mesmo se o fullscreen falhar ou nao for suportado.
3. O fluxo de save e criacao da run permanece intacto.
4. O build continua valido apos a entrega.
