# TASK-002 - Criar um Componente de Tooltip Estilizado para Cartas

## Tipo

Task

## Status

Concluido

## Objetivo

Criar um tooltip reutilizavel em formato de card estilizado para mostrar informacoes detalhadas da carta.

## Conclusao

- componente `CardTooltip` criado
- tooltip com layout, seta, metadados e destaque de rendimento

## Validacao em Execucao

- ao passar o mouse sobre uma carta, deve aparecer um card flutuante estilizado
- esse card deve mostrar nome, explicacao, efeito base e rendimento
