## Tipo

Historia

## Vinculo

## Declaracao

Como jogador, quero visualizar o estado do save e poder limpar esse progresso local, para validar a persistencia e controlar o fluxo de continuidade.

## Criterios de Aceitacao

1. `Opcoes` mostra o estado do save.
2. Existe uma acao para limpar o save.
3. A UI reflete quando o save deixa de existir.
