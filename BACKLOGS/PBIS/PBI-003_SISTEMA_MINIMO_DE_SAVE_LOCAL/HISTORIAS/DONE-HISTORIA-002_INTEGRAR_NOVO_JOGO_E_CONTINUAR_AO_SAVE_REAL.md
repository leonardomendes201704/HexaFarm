## Declaracao

Como jogador, quero que `Novo Jogo` e `Continuar` utilizem dados salvos reais, para perceber que a continuidade do jogo ja tem base persistente.

## Criterios de Aceitacao

1. `Novo Jogo` cria um save real.
2. A home reflete metadados do save existente.
3. `Continuar` exibe informacoes persistidas da run.
