## Declaracao

Como base tecnica do jogo, quero um modelo minimo e versionado de save local, para sustentar continuidade e evolucao do estado salvo sem depender de um formato temporario.

## Criterios de Aceitacao

1. Existe um contrato de save versionado.
2. O save pode ser lido e escrito com seguranca.
3. O formato stub anterior pode ser migrado.
