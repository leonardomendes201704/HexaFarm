# PBI-051 - Adicionar Atalho para Ocultar Slabs 3D

## Tipo

PBI

## Status

Concluido

## Epic Vinculado

- `EPICO-037 - Renderizacao 3D Isometrica Real do Stage`

## Objetivo

Permitir ocultar e reexibir rapidamente os slabs superficiais do stage 3D para depuracao visual durante a run.

## Escopo

- adicionar um atalho de teclado para alternar a visibilidade dos slabs
- aplicar o toggle na renderizacao dos acentos de superficie dos tiles 3D
- documentar o novo atalho na ajuda da run
