# TASK-007 - Implementar Estado do Botao Continuar

## Tipo

Task

## Vinculo

- Epico: `EPICO-001`
- PBI: `PBI-001`
- Historia: `HISTORIA-003`

## Objetivo

Fazer o botao `Continuar` responder ao estado retornado pelo adaptador de save.

## Descricao Tecnica

- renderizar o botao `Continuar`
- habilitar ou desabilitar conforme save local
- preparar o evento de clique para integracao futura

## Saida Esperada

O usuario entende visualmente se ha ou nao uma sessao disponivel para continuar.

## Definicao de Pronto

A task estara pronta quando `Continuar` refletir corretamente a existencia de save valido.
