# TASK-005 - Conectar Acao de Novo Jogo

## Tipo

Task

## Vinculo

- Epico: `EPICO-001`
- PBI: `PBI-001`
- Historia: `HISTORIA-002`

## Objetivo

Conectar o botao `Novo Jogo` a um handler inicial, rota ou estado de transicao previsivel.

## Descricao Tecnica

- definir evento de clique
- apontar para fluxo real ou placeholder consistente
- manter o contrato pronto para integracao futura com inicio de run

## Saida Esperada

Ao clicar em `Novo Jogo`, a aplicacao executa uma transicao valida e controlada.

## Definicao de Pronto

A task estara pronta quando `Novo Jogo` deixar de ser apenas visual e passar a executar um fluxo inicial consistente.
