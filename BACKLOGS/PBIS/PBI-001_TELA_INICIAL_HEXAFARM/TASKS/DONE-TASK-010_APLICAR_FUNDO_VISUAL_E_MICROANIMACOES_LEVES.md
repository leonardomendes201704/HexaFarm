# TASK-010 - Aplicar Fundo Visual e Microanimacoes Leves

## Tipo

Task

## Vinculo

- PBI: `PBI-001`
- Historia: `HISTORIA-001`

## Objetivo

Adicionar um nivel inicial de atmosfera visual para evitar uma tela generica e reforcar a identidade do jogo.

## Descricao Tecnica

- criar fundo coerente com o universo de `HexaFarm`
- aplicar efeitos leves de movimento ou transicao
- preservar boa performance e clareza da interface

## Saida Esperada

A tela inicial ganha identidade visual sem virar uma implementacao pesada ou desnecessariamente complexa.

## Definicao de Pronto

A task estara pronta quando a tela estiver visualmente mais caracteristica e continuar leve no browser.
