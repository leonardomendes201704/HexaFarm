# HISTORIA-002 - Implementar Acoes Principais de Entrada

## Tipo

Historia

## Vinculo

- Epico: `EPICO-001`
- PBI: `PBI-001`

## Declaracao

Como jogador, quero ver e usar as acoes principais da tela inicial, para iniciar uma nova partida ou acessar opcoes basicas sem friccao.

## Objetivo

Disponibilizar as acoes principais da entrada do jogo com hierarquia clara, comportamento previsivel e base pronta para integracao com o fluxo futuro.

## Escopo

- exibir `Novo Jogo`
- exibir `Configuracoes` ou `Opcoes`
- estruturar o menu principal

## Criterios de Aceitacao

1. O botao `Novo Jogo` aparece com destaque.
2. O botao `Novo Jogo` pode ser acionado e dispara um handler valido.
3. O usuario visualiza `Configuracoes` ou `Opcoes`.
4. O acesso a configuracoes pode usar um fluxo stub sem bloquear a navegacao.

## Definicao de Pronto

A historia estara pronta quando as acoes principais estiverem visiveis, clicaveis e tecnicamente preparadas para evolucao.
