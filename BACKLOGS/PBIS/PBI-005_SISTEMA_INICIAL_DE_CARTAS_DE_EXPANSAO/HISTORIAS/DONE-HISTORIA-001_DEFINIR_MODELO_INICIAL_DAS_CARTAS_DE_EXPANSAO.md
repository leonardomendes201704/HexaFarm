## Declaracao

Como base tecnica do prototipo, quero um conjunto inicial de cartas de expansao com tipos claros, para que o mapa cresca em funcao de cartas e nao de um botao generico.

## Criterios de Aceitacao

1. Existe um modelo de carta de expansao.
2. Cada carta aponta para um tipo de tile.
3. Existe um conjunto inicial de cartas para o prototipo.
