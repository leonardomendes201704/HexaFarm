# HISTORIA-003 - Permitir Expansao Simples de Tiles e Refletir no Save

## Tipo

Historia

## Vinculo

- PBI: `PBI-004`

## Declaracao

Como jogador, quero usar uma acao simples de expansao e adicionar um tile novo ao mapa, para validar o loop de crescimento espacial da fazenda.

## Criterios de Aceitacao

1. Existe uma acao de expansao acionavel.
2. Um slot de fronteira valido pode virar tile colocado.
3. O save reflete o numero de tiles apos a expansao.
